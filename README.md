# Globetrotter – Travel Exploration App

Globetrotter is a travel exploration web application built using **React** and **Vite**.  
It allows users to explore popular travel destinations through visually rich cards and navigate to detailed trip pages.

---

## Tech Stack
- **Frontend:** React
- **Bundler:** Vite
- **Languages:** JavaScript, CSS
- **Design:** Responsive layout with reusable components

---

## Features Implemented
- Destinations page with interactive destination cards
- Trip details page for each destination
- Component-based architecture for reusable UI elements
- Responsive design for different screen sizes
